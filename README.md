# 📚 Simple Library Management

A responsive and user-friendly **Library Management Web Application** built using **HTML, CSS, and JavaScript**. This project allows users to browse, search, and filter books efficiently. It is designed as a portfolio project to demonstrate front-end development skills.

---

## 👩‍💻 Developed By
**Rudrakshi Rathore**

---

## 📌 Problem Statement
Build a simple library management page where:

- 📖 **View Books**
  - Users can see a list of available books.
  - Instead of loading all results at once, paging can be implemented (e.g., 10 results per page) or infinite scroll for enhanced performance.

- 🔍 **Filter & Search**
  - Users can filter books based on:
    - **Title**
    - **Author**
    - **Subject**
    - **Publish Date**

- 📊 **Book Count**
  - Users can view the count of books based on each filtering criterion.

---

## 🚀 Features
- 🔎 Search books by title.
- 📚 Filter books by subject/category.
- 👩‍💼 Display author and publication date.
- 📱 Fully responsive design.
- 🎨 Clean and modern UI.
- ⚡ Dynamic rendering of books using JavaScript.
- 🌐 Easy deployment using GitHub and Netlify.
